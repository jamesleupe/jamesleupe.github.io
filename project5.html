Leaf Generator

Project Description

Custom unity window for placing a prefab leaf mesh over a source mesh.

Project Details

  • Spawns leaves on each face of a source mesh (such as a sphere).
  • Allows random rotation of the leaves.
  • Bake the final result into one mesh.
  • Normals are copied from the source mesh to the leaf billboards to create similar shading as the source mesh.
