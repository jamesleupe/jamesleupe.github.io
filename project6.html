Game Engine

Project Description

Basic 2D/3D Engine built on the MonoGame framework.

Project Details

  • Special effects including lightning and particle systems.
  • Game window runs embedded in Windows Forms.
  • Menu system and many UI controls implemented (TextBox, CheckBox, Button, Label, Slider)
  • Farseer Physics integration.
  • World Builder for building (placing 3D objects) and saving scenes.
  • Scenes saved in a serialized binary format.
