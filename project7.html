3D Midi Visualizer

Project Description

MIDI (Musical Instrument Digital Interface) is a standard used for representing musical instrument input in digital form. This unity programs converts multiple midi data into a 3D representation.

Project Details

  • Can read and load multiple MIDI files.
  • The notes are moved visually according to the tempo set in the midi file. (Had to convert time signature to distance moved per frame).
  • Post processing glow effect applied when notes are started.
  • Does not produce audio itself, only 3D representation.
