Barrier and Force Fields for Unity

Project Description

This project was an exercise in shader writing within the unity engine.

Project Details

  • Vert/Frag shaders written in HLSL/Cg
  • Multiple activation/deactivation methods including height scan, fade in and dissolve.
  • Cylindrical barrier - Useful for battle royale style games.
  • Animate textures over the objects surface.
