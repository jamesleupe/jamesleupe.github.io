Grass shader (geom + tessellation stages)

Project Description

This project demonstrates the use of geometry and tessellation shaders in the generation of grass blades over a terrain mesh in Unity. Progress pics at bottom of page.

Project Details

  • Vert/Geom/Tess/Frag shaders all written in HLSL/Cg
  • Script divides the grass into chunks to speed up processing (hide chunks not in line of sight)
  • Shadows not implemented but keeps performance up.
  • Supports density map, height map (height of grass over different areas) and basic collision.
  • Support for multiple LOD levels (reduces triangle count based on distance to camera).
  • Grass blades generated entirely on GPU in geometry shader
